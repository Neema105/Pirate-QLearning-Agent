My model focused on designing and training a reinforcement learning agent using Q-learning to navigate a maze environment and make optimal decisions through trial-and-error learning.

For this project, I was provided with starter code that defined the maze environment and the state space. I wrote and refined the Q-learning components for the agent myself, including the logic for exploration versus exploitation, updating Q-values, managing experience memory, and tuning training parameters. I also tested the agent’s performance and adjusted the code to improve how effectively it learned the optimal path through the maze.

I approach problems by breaking them down into smaller, manageable parts, designing a logical solution, and then testing and refining that solution through iteration. In this project, that meant experimenting with algorithms, analyzing results, and making improvements based on how the system behaved.
