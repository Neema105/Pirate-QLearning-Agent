# CS370-Pirate-Agent
My model focused on designing and training a reinforcement learning agent using Q-learning to navigate a maze environment and make optimal decisions through trial-and-error learning.

Briefly explain the work that you did on this project: What code were you given? What code did you create yourself?
For this project, I was provided with starter code that defined the maze environment, state space, and the overall structure for training a reinforcement learning agent. I wrote and refined the Q-learning components myself, including the logic for exploration versus exploitation, updating Q-values, managing experience memory, and tuning training parameters. I also tested the agent’s performance and adjusted the code to improve how effectively it learned the optimal path through the maze.
